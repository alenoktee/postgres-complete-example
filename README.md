# PostgreSQL Complete Example

Это полнофункциональный пример PostgreSQL проекта с реальными данными.

## Структура

- `init.sql` - Создание таблиц и вставка начальных данных
- `test.sql` - 10 различных SQL запросов для анализа данных
- `employees.csv` - Пример CSV файла с данными сотрудников
- `README.md` - Этот файл

## Что содержит init.sql?

- **employees** - таблица сотрудников (8 записей)
- **projects** - таблица проектов (5 записей)
- **assignments** - таблица назначений (10 записей)
- Индексы для оптимизации
- Внешние ключи для целостности данных

## Что содержит test.sql?

10 готовых SQL запросов:

1. **Базовая статистика** - количество записей в каждой таблице
2. **Анализ по отделам** - зарплаты, количество сотрудников
3. **Топ 5 высокооплачиваемых** - лучше платящие должности
4. **Статус проектов** - обзор проектов по статусам
5. **Использование сотрудников** - на сколько проектов назначены
6. **Состав команд проекта** - кто на каком проекте работает
7. **Статус сотрудников** - активные vs неактивные
8. **Распределение зарплат** - диапазоны зарплат
9. **График проектов** - сроки начала и конца
10. **Зарплатная ведомость** - по отделам

## Как использовать?

### На компьютере:

```bash
# Клонируйте репозиторий
git clone https://github.com/YOUR_USERNAME/postgres-complete-example.git

# Заходите в папку
cd postgres-complete-example

# Смотрите файлы
cat init.sql
cat test.sql
```

### Через API:

```powershell
# PowerShell
$testingId = @{
    language = "postgresql"
    repositoryUrl = "https://github.com/YOUR_USERNAME/postgres-complete-example.git"
    branch = "main"
} | ConvertTo-Json | Invoke-WebRequest -Uri "http://localhost:5000/api/Testing" -Method POST | Select-Object -ExpandProperty Content

# Через 5 секунд проверьте результаты
Invoke-WebRequest -Uri "http://localhost:5000/api/Testing/$testingId/results"
```

## Данные в примере

### Сотрудники
- 8 сотрудников из разных отделов
- Зарплаты от $65k до $95k
- Различные должности (Senior, Manager, QA и т.д.)

### Проекты
- 5 проектов в разных статусах
- Бюджеты от $50k до $150k
- Сроки выполнения от 1 до 6 месяцев

### Назначения
- 10 назначений сотрудников на проекты
- Разные роли (Tech Lead, Developer, Manager и т.д.)
- Разный объём часов работы

- Статистику по всем 10 запросам
- Структурированные результаты с заголовками
- Таблицы с анализом данных
